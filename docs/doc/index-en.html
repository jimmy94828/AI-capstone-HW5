A group-authored ontology for grounding course task objects, task roles, manipulation affordances, and inferred graspability.

graspable objectc back to ToC or Class ToC

IRI: https://hcis.io/ontology/aicapstone/2026/GraspableObject

A physical object inferred to be graspable because it has at least one grasping affordance.
A physical object inferred to afford grasping by a robot end effector in the task context.
is equivalent to
physical object c and (has affordance op some grasping affordance c)
